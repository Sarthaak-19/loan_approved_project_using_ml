# 🏦 Loan Approval Prediction using Machine Learning

A machine learning project that predicts whether a loan application will be **Approved** or **Rejected** based on applicant information. This project demonstrates the complete machine learning workflow, from data preprocessing to model evaluation.

---

## 📌 Project Overview

This project includes:

- Data Cleaning
- Handling Missing Values
- Label Encoding
- One-Hot Encoding
- Feature Scaling using StandardScaler
- Model Training
- Model Evaluation
- User Input Prediction

---

## 📊 Dataset

- **Dataset:** Loan Approval Prediction Dataset
- **Source:** https://github.com/prasertcbs/basic-dataset/blob/master/Loan-Approval-Prediction.csv
